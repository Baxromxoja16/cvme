## 3. Technical SEO & Multilingual Structure

### URL Structure Recommendation

**Use Subfolders: `cvme.uz/uz/`, `cvme.uz/ru/`, `cvme.uz/en/`**

- **Why?** It consolidates Domain Authority. Every backlink to `/en/` also boosts the power of the main domain for `/uz/`. Subdomains (`en.cvme.uz`) distribute authority, making it harder to rank everything.
- **Hreflang**: Implement `rel="alternate" hreflang="x"` in the `<head>` of every page to tell Google exactly which version to show to which user.

### Wildcard Subdomain Strategy (`*.cvme.uz`)

To prevent "Spam" flags and boost the main domain:

1. **Quality Filter**: Only allow indexing (`index, follow`) for profiles that have at least 80% completion (Avatar + Contacts + Experience). Empty profiles get `noindex`.
2. **Schema Markup**: Every user profile MUST have `Person` and `ProfilePage` JSON-LD schema. Google loves structured data.
3. **Cross-Linking**: The main `cvme.uz` home page should have a "Featured Members" or "Public Directory" section linking to high-quality user subdomains. This passes juice from core to subdomains.
